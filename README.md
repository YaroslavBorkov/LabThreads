# Лаб №2 Параллельные вычисления. Шаблон Producer -> Consumer

## Постановка задачи (Вар 2)
Входные данные - текстовый файл со списком контактов.  
Формат строки: "Фамилия;Имя;Отчество;Телефон"

1. Прочитать входной файл.
2. Параллельно обработать записи.
3. Сгруппировать контакты по первой букве фамилии (ASCII).
4. Сохранить результат в выходной файл.

## Архитектура решения
Используется конвейер из трёх стадий:
1. **Producer (1 поток)** - читает файл построчно, парсит строки и помещает задачи в блокирующую очередь
2. **Consumers (N потоков)** - извлекают задачи из очереди, выполняют обработку и помещают результаты во вторую очередь
3. **Aggregator (1 поток)** - собирает результаты обработки в хеш-таблицу и записывает итог в файл

Для синхронизации используется собственная реализация блокирующей очереди с ограниченной ёмкостью

## Используемые технологии
- Многопоточность: `std::thread`,`std::mutex`,`std::condition_variable`
- Паттерн: Producer -> Consumer
- Сборка CMake
